# tweakcn Preset Reference

Quick reference for tweakcn theme presets and their characteristics.

## Dark Themes

| Preset | Style | Best For |
|--------|-------|----------|
| **graphite** | Deep slate grays, professional | Dev tools, dashboards |
| **darkmatter** | Near-black, minimal | Terminal apps, code editors |
| **cosmic night** | Deep blue undertones | AI products, modern apps |
| **mono** | Pure black/white | Minimal, typography-focused |
| **claude** | Warm dark, AI-inspired | AI tools, chat apps |
| **midnight bloom** | Dark with purple accent | Creative tools |
| **caffeine** | Dark warm browns | Productivity, notes |

## Light Themes

| Preset | Style | Best For |
|--------|-------|----------|
| **modern minimal** | Clean white, subtle grays | General purpose |
| **clean slate** | Crisp, high contrast | Documentation, content |
| **notebook** | Paper-like, warm whites | Note apps, writing tools |
| **soft pop** | Light with soft colors | Consumer apps |
| **vercel** | High contrast, bold | Marketing, dashboards |

## Selection Criteria

1. **Product type** - Match the preset to your product's domain
2. **User preference** - Power users prefer dark, general users prefer light
3. **Content type** - Text-heavy apps need high contrast
4. **Brand alignment** - Choose colors that fit your brand

## Customization

After selecting a preset:

1. Adjust primary color to match brand
2. Fine-tune border opacity for your taste
3. Check contrast ratios for accessibility
4. Test both light and dark variants
